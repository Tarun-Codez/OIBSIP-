# Guess the Number

A small Java project where the computer picks a random number from 1 to 100 and the player tries to guess it using dialog boxes.

## Features

- Random number between 1 and 100
- Dialog-based input and feedback
- Limited attempts per round
- Multiple rounds
- Score based on how quickly the number is guessed

## How to run

Compile:

```text
javac GuessTheNumber.java
```

Run:

```text
java GuessTheNumber
```
